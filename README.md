# Classic CS Papers

> Reading the foundational papers of computer science, modelling the ideas in Python, and publishing notes on Medium.

The code here is not production software. It exists to make abstract ideas concrete. Every file documents which section and page of the original paper it draws from.

## Getting Started

Each paper folder has its own `README.md`, you can start there. The code files are standalone Python scripts with no external dependencies.

```bash
git clone https://github.com/niluzunoglu/classic-cs-papers
cd classic-cs-papers/turing/on-computable-numbers-1936
python section3_example1.py
```
